PAGINA 2 WEB

Una página web es un documento digital que forma parte de la World Wide Web y tiene una dirección única que le permite ser identificada y accesible en Internet. Estas páginas son entregadas por un servidor web al usuario y se visualizan a través de un navegador, que actúa como una herramienta de recuperación para acceder a la información que contienen.


Entre los tipos más comunes de páginas web se encuentran las tiendas en línea (e-commerce), los blogs personales, los sitios corporativos, los motores de búsqueda, las redes sociales, y las aplicaciones web como el correo electrónico y las wikis. También existen portales de contenido, portfolios de artistas, plataformas de video y foros de discusión, cada uno con un propósito y una audiencia específicos.

CLIENTE / SERVIDOR



El modelo cliente/servidor ha transformado la manera en que se desarrollan las aplicaciones distribuidas. En este enfoque, el cliente realiza solicitudes de servicios que son atendidas por el servidor, creando una interacción dinámica y eficiente. Esta arquitectura permite una clara separación entre la interfaz de usuario y la lógica de la aplicación, lo que no solo mejora la usabilidad, sino que también facilita la escalabilidad y la gestión de recursos.


Capas del Modelo

  • Presentación: La capa encargada de la interfaz y la interacción con el usuario, donde se muestran los datos de forma visual y accesible.
  • Lógica: Esta capa gestiona la lógica de negocio y el procesamiento de las solicitudes, actuando como un puente entre la presentación y los datos.
  • Datos: Aquí es donde se almacenan y gestionan los datos, garantizando su integridad y seguridad.
