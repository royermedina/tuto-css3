Tutorial de HTML5/HTML6

1. Introducción

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Desde su creación en 1993, HTML ha evolucionado significativamente para adaptarse a las crecientes necesidades de la web moderna.

HTML5, lanzado oficialmente en 2014, trajo consigo una revolución en el desarrollo web, introduciendo nuevas etiquetas semánticas, mejoras en formularios, y soporte nativo para contenido multimedia. Por su parte, HTML6, aunque aún no es un estándar oficial, promete continuar esta evolución con mejoras en áreas como la accesibilidad, la semántica y la integración con tecnologías emergentes.

2. Evolución de HTML

La evolución de HTML ha sido un viaje fascinante:

  • HTML 1.0 - 4.01: Establecieron las bases del lenguaje HTML, permitiendo la creación de documentos web con elementos básicos como párrafos, enlaces, imágenes y tablas. HTML 4.01 introdujo características más avanzadas, como el uso de hojas de estilo (CSS) para la presentación y scripts (JavaScript) para la interactividad, haciendo que las páginas web fueran más dinámicas y estilizadas.
  • XHTML: Fue un intento de hacer HTML más estricto y XML-compatible, lo que implicaba seguir reglas de sintaxis más rígidas, como el uso de etiquetas cerradas y atributos en minúsculas. Esto permitió una mejor estructuración del contenido y facilitó la transición a futuras tecnologías web, asegurando la compatibilidad con dispositivos más modernos.
  • HTML5: Representó un gran salto adelante al introducir nuevas etiquetas semánticas (header, nav, article, section, etc.), permitiendo una estructura más clara y comprensible del contenido web. Además, añadió APIs para gráficos (canvas y SVG), multimedia (audio y video), y almacenamiento local, y trajo importantes mejoras en formularios y elementos interactivos, haciéndolos más eficientes y funcionales sin necesidad de plugins externos.
  • HTML6 (en desarrollo): Aunque aún no es oficial, se espera que HTML6 incluya mejoras significativas en cuanto a accesibilidad y seguridad, así como una integración más profunda con tecnologías emergentes como la realidad virtual (VR), realidad aumentada (AR) y la WebXR. También se anticipa la optimización de la experiencia del usuario en dispositivos móviles y el soporte para nuevas características de inteligencia artificial.

3. Formularios

HTML5 revolucionó los formularios web introduciendo nuevos tipos de input y validación del lado del cliente. Esto mejora la experiencia del usuario y reduce la carga en el servidor. Los formularios de HTML5 incorporan nuevos tipos de datos (input) como fechas, colores, direcciones de correo electrónico, rangos, etc., y la autovalidación de datos sin JavaScript. Al mismo tiempo, veremos todas las propiedades y eventos CSS3 (como [required] o :invalid) que permiten configurar cualquier elemento o respuesta de autovalidación de un formulario.

4. Listas

HTML ofrece tres tipos principales de listas: ordenadas (<ol>), no ordenadas (<ul>) y de definición (<dl>). Bootstrap proporciona estilos adicionales para mejorar su apariencia.


        <!-- Lista no ordenada -->
        <ul class="list-group">
            <li class="list-group-item">Elemento 1</li>
            <li class="list-group-item">Elemento 2</li>
            <li class="list-group-item">Elemento 3</li>
        </ul>

        <!-- Lista ordenada -->
        <ol class="list-group list-group-numbered">
            <li class="list-group-item">Primer paso</li>
            <li class="list-group-item">Segundo paso</li>
            <li class="list-group-item">Tercer paso</li>
        </ol>

        <!-- Lista de definición -->
        <dl class="row">
            <dt class="col-sm-3">HTML</dt>
            <dd class="col-sm-9">HyperText Markup Language</dd>
            <dt class="col-sm-3">CSS</dt>
            <dd class="col-sm-9">Cascading Style Sheets</dd>
        </dl>

5. Columnas de texto

Bootstrap facilita la creación de diseños de múltiples columnas utilizando su sistema de rejilla. Además, CSS3 proporciona propiedades específicas para dividir el texto en columnas.


        <!-- Usando el sistema de rejilla de Bootstrap -->
        <div class="row">
            <div class="col-md-6">
                <p>Contenido de la primera columna...</p>
            </div>
            <div class="col-md-6">
                <p>Contenido de la segunda columna...</p>
            </div>
        </div>

        <!-- Usando CSS3 multi-column layout -->
        <style>
        .multi-column-text {
            column-count: 2;
            column-gap: 40px;
        }
        </style>
        <div class="multi-column-text">
            <p>Texto que se distribuirá en múltiples columnas...</p>
        </div>

6. Multimedia (Video y Audio)

HTML5 introdujo las etiquetas <video> y <audio> para integrar contenido multimedia de manera sencilla.


        <!-- Video -->
        <video width="320" height="240" controls>
            <source src="video.mp4" type="video/mp4">
            Tu navegador no soporta la etiqueta de video.
        </video>

        <!-- Audio -->
        <audio controls>
            <source src="audio.mp3" type="audio/mpeg">
            Tu navegador no soporta la etiqueta de audio.
        </audio>

7. Sombras, Transparencias y Degradados

CSS3 permite crear sombras, transparencias y degradados para mejorar la estética de las páginas web.


        <style>
        .box {
            width: 200px;
            height: 200px;
            background-color: rgba(0, 0, 255, 0.5);
            box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5);
            background: linear-gradient(to right, red, yellow);
        }
        </style>

        <div class="box"></div>

8. Animaciones y Transiciones

CSS3 introduce las técnicas de animación y transiciones para crear interacciones y animaciones en las páginas web. Muy parecidas a las transiciones (transition) del módulo anterior, vamos a ver la funcionalidad y las diferencias que existen entre las transiciones y las animaciones (animation), entre las que se pueden resaltar una mayor posibilidad en la definición de movimientos, la visualización sin intervención del usuario (al entrar en la página), fijar un retardo en el inicio o la reproducción indefinida de la animación (loop).
